## Features

* 로그인 로그아웃 구현 
* 회원가입 시 중복 방지, 이메일 인증기능
* 로그인 시 sns 로그인, 비밀번호 찾기 기능
* 각각 게시판 카테고리로 매핑, 페이징 적용 완료, 각각 게시판 별 검색기능, 필터기능
* 노래 업로드 게시판에 파일 업로드 및 노래 재생 가능
* 공지사항 게시판은 ADMIN만 작성가능(Spring Security 이용해서 각 유저 등급 조정가능)
* 메인페이지에 각각 게시판 좋아요로 필터링
*
